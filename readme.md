# 📱 minisand-Kplus-listener (Android Background Service)

แอปพลิเคชัน Android ที่พัฒนาด้วย Kotlin ทำหน้าที่เป็น **Background Service** เพื่อดักจับการแจ้งเตือน (Notification) จากแอปพลิเคชัน **K PLUS (ธนาคารกสิกรไทย)** แบบเรียลไทม์ เมื่อตรวจพบการโอนเงินเข้า แอปจะทำการสกัดยอดเงินและส่งข้อมูลผ่าน API (Webhook) ไปยังเซิร์ฟเวอร์ของคุณโดยอัตโนมัติ

## 🚀 คุณสมบัติ
- **Real-time Detection:** ตรวจจับแจ้งเตือนเงินเข้าทันที ไม่ต้องเปิดแอปค้างไว้
- **Smart Regex Extraction:** กรองและดึงเฉพาะ "ตัวเลขยอดเงิน" ออกจากข้อความแจ้งเตือนของ K PLUS ได้อย่างแม่นยำ
- **Secure Webhook:** ส่งข้อมูลไปยังเซิร์ฟเวอร์ปลายทางผ่าน HTTP POST พร้อมระบบ API Key ยืนยันตัวตน
- **Background Persistence:** ทำงานได้ต่อเนื่องแม้ปิดหน้าจอ (เมื่อตั้งค่า Battery Optimization ถูกต้อง)

## 🛠 วิธีการทำงาน
1. เซิร์ฟเวอร์ของคุณเตรียม API Endpoint รอรับข้อมูล JSON
2. แอป **K PLUS** ในมือถือ (เครื่องที่ติดตั้งแอปนี้) เด้งการแจ้งเตือนว่า *"ได้รับเงินโอน..."*
3. Service `KplusNotificationListener` ตรวจจับพบ Package Name `com.kasikorn.retail.mbanking.mobile`
4. อ่านข้อความ (Text) จากแจ้งเตือน -> สกัดยอดเงิน -> ยิง API ไปยังเซิร์ฟเวอร์ทันที

## 📦 โครงสร้างโปรเจกต์
- `KplusNotificationListener.kt`: หัวใจหลักของแอป เป็น Background Service ที่คอยฟังและอ่านข้อความจากการแจ้งเตือน พร้อมโค้ดยิง HTTP POST ไปยังเซิร์ฟเวอร์
- `MainActivity.kt`: หน้าจอหลักสำหรับขอสิทธิ์การเข้าถึงแจ้งเตือน และมีปุ่มทดสอบยิง API

## ⚙️ การตั้งค่าก่อนใช้งาน
**ในไฟล์ `KplusNotificationListener.kt` และ `MainActivity.kt`**
1. เปลี่ยน `URL` ให้เป็น API ของคุณ
2. เปลี่ยน `X-API-Key` ให้ตรงกับเซิร์ฟเวอร์ของคุณ

## ⚠️ ขั้นตอนสำคัญเพื่อให้แอปทำงานได้ 100%
เพื่อให้ Android ไม่สั่งฆ่าแอป (Kill Process) เมื่อทำงานเบื้องหลังนานๆ ต้องตั้งค่าดังนี้:

1. **สิทธิ์การเข้าถึงการแจ้งเตือน (Notification Access):**
    - เมื่อเปิดแอปครั้งแรก แอปจะพาคุณไปยังหน้าต่างตั้งค่า ให้หาชื่อแอป `minisand-Kplus-listener` และกด "อนุญาต"
2. **ปิดการประหยัดแบตเตอรี่ (Battery Optimization):**
    - มือถือรุ่นใหม่มักจะปิดแอปที่ทำงานเบื้องหลัง ให้ไปที่ `การตั้งค่า > แอป > minisand-Kplus-listener > แบตเตอรี่` เลือกเป็น **"ไม่จำกัด (Unrestricted)"**
3. **ล็อคแอปในหน้า Recent Apps:**
    - กดค้างที่แอป `minisand-Kplus-listener` แล้วเลือกปุ่ม **รูปแม่กุญแจ (Lock)** เพื่อบอกระบบว่าห้ามปัดแอปนี้ทิ้ง
